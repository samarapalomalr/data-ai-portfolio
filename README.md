# Data Engineering & AI Portfolio

Portfólio web focado em **Engenharia de Dados, Machine Learning e Inteligência Artificial**.

O objetivo deste projeto é apresentar projetos desenvolvidos em ciência de dados, competições Kaggle e aplicações de IA.

---

# Tecnologias utilizadas

Frontend

- JavaScript
- React / Next.js
- CSS

Data & AI

- Python
- Pandas
- NumPy
- Scikit-learn
- Machine Learning

Ferramentas

- Git
- VSCode
- Kaggle

---

# Funcionalidades

Página inicial com apresentação
Seção de tecnologias
Lista de projetos
Página sobre
Página de contato
Layout moderno em tema escuro

# Como rodar o projeto

Clone o repositório
git clone https://github.com/samarapalomalr/data-ai-portfolio.git

Execute o projeto
npm run dev
